CSE undergrad with a heavy bias for shipping. I spend most of my time building **AI agents** — for code, for music, for medical data, for browser automation — and contributing to the libraries that make them possible. Looking for hard problems, good collaborators, and reasons to learn something new.

### Pinned — what I'm building right now

| Project | What it does |
|---|---|
| [**moozix**](https://github.com/ShivamShrivastava18/moozix) | Cross-user Spotify compatibility engine — find out how musically compatible you are with someone, with hard data and a witty narrative. |
| [**AI-Agent-for-Synthetic-Data-Generation**](https://github.com/ShivamShrivastava18/AI-Agent-for-Synthetic-Data-Generation) | Production API that generates conditional synthetic tabular rows (Gaussian Copula / Conditional Histogram), validates fidelity with TSTR / KS / JS, and exposes a Groq-powered conversational agent. Dockerized, FastAPI, frontend included. |
| [**github-battle-bot**](https://github.com/ShivamShrivastava18/github-battle-bot) | Discord bot that compares friends' GitHub stats and roasts them — `/battle`, `/stats`, `/roast` slash commands powered by GitHub API + Groq. |
| [**strudel_agent**](https://github.com/ShivamShrivastava18/strudel_agent) | AI agent that writes [Strudel](https://strudel.cc) code to compose live music. |
| [**persona-driven-e2e-testing**](https://github.com/ShivamShrivastava18/persona-driven-e2e-testing) | Claude Code skill that runs full customer + admin E2E QA as one autonomous pass — discovers the app, generates personas, dispatches tester subagents, verifies cross-surface state. |
| [**LuciferDono/contribute**](https://github.com/LuciferDono/contribute) | Contributor — Claude Code plugin for full-lifecycle OSS contribution. 12 phases from issue discovery to PR merge in one `/contribute` command. |
